##  Step-by-Step Guide

### 1.  Create Your HTML File

Create a file called `index.html` on your computer with some basic content.

Example:

```html
<!DOCTYPE html>
<html>
<head>
    <title>My First GitHub Page</title>
</head>
<body>
    <h1>Hello, GitHub Pages!</h1>
    <p>This is my first static website.</p>
</body>
</html>
````

---

### 2.  Push to a New GitHub Repository

1. Go to [github.com](https://github.com) and create a **new repository**.

   * Name it something like `protfolio-static-webpage`
   * Set it to **public**
2. Open your terminal and run the following:

```bash
git init
git add .
git commit -m "Initial commit"
git branch -M main
git remote add origin https://github.com/Gaurav1517/protfolio-static-webpage.git
git push -u origin main
```

>  Replace `your-username` with your actual GitHub username.

---

### 3.  Enable GitHub Pages

1. Go to your **repo on GitHub**
2. Click on **Settings**
3. Scroll down to **Pages** (or go to the “Pages” section from the left menu)
4. Under **"Source"**, select:

   * **Branch**: `main`
   * **Folder**: `/ (root)`
5. Click **Save**

---

### 4.  Access Your Live Website

After a few seconds, GitHub will show a link like:

```
https://gaurav1517.github.io/protfolio-static-webpage/
```

Click it — your site is now live!
